# Hy-Fi-Death-Run
Hy-Fi Death Run recreates the classic runner-vs-trapper experience inside Hytale. One team races through deadly obstacles while another triggers traps. Includes round timers, map rotations, role selection, and matchmaking options. Designed for replayability on minigame servers.
